My Projects.

TETRIS

I developed a Tetris clone alongside Minseo Kim as part of a final project for Northwestern's CS 211 class. This clone was developed using C++ and the ge211 game engine. We designed the graphical user interface and developed the gameplay using the typical user commands for Tetris: left and right arrow to move sideways, down arrow to speed up drop, space bar to auto drop, and shift to hold a block. Game displays number of lines cleared to allow users to view their performance.

LET'S GROW

Let's Grow Alameda is a community organization focused on promoting mental health and wellness through gardening. We worked in conjunction with elementary and middle schools in school district to provide access to seeds. As 1 of 2 web developers for the site, I was tasked with designing and developing the homepage and grow page, which features gardening tips and the organization's Youtube tutorials. This project uses HTML, CSS, and Javascript.
